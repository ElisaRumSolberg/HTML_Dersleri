# 🔐 Login UI Design

This project is a **responsive and modern login form** created using only HTML and CSS.

## 📌 Features

- Responsive layout
- Clean and modern design
- Gradient background
- Styled input fields and buttons
- Hover and focus effects
- "Remember Me" checkbox
- "Forgot Password" and "Sign Up" links

## 💡 Technologies Used

- HTML5
- CSS3 (Flexbox, Gradients, Transitions)

## 🎯 What I Learned

- Designing form inputs and labels with CSS
- Using gradients for background styling
- Improving user experience with `:hover` and `:focus`
- Structuring and centering a form using Flexbox
- Making forms responsive for all screen sizes
